# Sable-Style Shader Exploration

## Purpose
Explore shader techniques inspired by the painterly and graphic style seen in *Sable*-like rendering.

## Visual Goals
- Flat-yet-rich color blocks
- Controlled gradients and soft transitions
- Stylized line/edge emphasis where needed

## Technical Directions
- Ramp-based diffuse and stylized shadow shaping
- Texture-driven detail layering with minimal noise
- Color grading and post-process support for cohesive look

## Outcome
A curated set of experiments that isolate and reproduce key components of a Sable-like visual language.
