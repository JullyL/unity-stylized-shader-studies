# Toon Fridge Material Study

## Purpose
Experiment with stylized material and lighting techniques for a fridge-focused scene/object study in Unity.

## Scope
- Shader and material look development
- Texture and color style iteration
- Lighting + post-process tuning for stylized results

## Suggested Workflow
1. Set baseline scene lighting and camera framing.
2. Iterate albedo/normal/roughness response for target style.
3. Add stylization controls (ramp, edge accent, contrast shaping).
4. Compare performance and visual quality across variants.

## Notes
Use this folder as an isolated sandbox for material experiments before merging reusable ideas into other projects.
