# HDRP Shader Practice Lab

## Purpose
Practice and prototype shader techniques specifically in Unity HDRP.

## Focus Areas
- HDRP material/shader graph workflows
- Lighting response under physically based settings
- Advanced rendering features (reflections, volumetrics, post effects)

## Suggested Experiments
1. Build baseline shader graph variants for surface types.
2. Test HDRP lighting scenarios (indoor/outdoor/high contrast).
3. Profile shader complexity and optimize expensive nodes.
4. Document reusable graph patterns for future projects.

## Output
A collection of HDRP shader studies and reusable material setups.
