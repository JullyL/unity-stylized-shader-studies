# URP Shader Practice Lab

## Purpose
Develop and test shader techniques in Unity URP with a balance of style and performance.

## Focus Areas
- URP Shader Graph fundamentals and custom node logic
- Lightweight stylized shading pipelines
- Mobile/real-time friendly optimization strategies

## Suggested Experiments
1. Create core stylized shaders (toon, gradient, mask-driven blends).
2. Validate visual consistency across URP render settings.
3. Measure frame cost for each shader feature.
4. Build a small reusable library of URP materials.

## Output
URP-ready shader experiments and references for later production projects.
